================================================================================
  GADGET STORE — GHID COMPLET DE TESTARE MANUALA
================================================================================

PREGATIRE
---------
1. Porneste backend-ul:  cd Gadget_Store.API  ->  dotnet run
2. Porneste frontend-ul: cd Gadget_Store.Web  ->  npm run dev
3. Deschide browserul la http://localhost:5173
4. Ai nevoie de 2 conturi: unul Customer (inregistrat nou) si unul Admin
   (creat prin seed sau schimbat rol din BD)


================================================================================
BLOC 1 — AUTENTIFICARE
================================================================================

[ ] 1.1  Inregistrare cont nou
         - Mergi la /register
         - Completeaza: firstName, lastName, email, parola (min 8 car.)
         - Apasa "Inregistreaza-te"
         - Asteptat: redirectat la /  , badge "Customer" vizibil in header

[ ] 1.2  Login cu date corecte
         - Delogheaza-te (daca esti autentificat)
         - Mergi la /login
         - Introdu email + parola corecte
         - Asteptat: redirectat la / , numele userului apare in header

[ ] 1.3  Login cu date gresite
         - Introdu parola gresita
         - Asteptat: mesaj de eroare toast, ramai pe /login

[ ] 1.4  Redirect GuestOnly
         - Fiind autentificat, acceseaza manual /login sau /register
         - Asteptat: redirectat automat la /

[ ] 1.5  Delogare
         - Apasa butonul de logout din header
         - Asteptat: token sters, redirectat la / , header arata "Login"

BLOC 2 — CATALOG SI NAVIGARE
================================================================================

         - Mergi la /catalog

[ ] 2.3  Filtrare dupa pret

         - Verifica ca lista se filtreaza corect

[ ] 2.4  Filtrare "Doar in stoc"

         - Verifica ordinea listei dupa fiecare optiune

[ ] 2.6  Navigare categorii

         - Verifica ca se filtreaza produsele
         - Testeaza si subcategorii (Composite pattern)

[ ] 2.7  Paginare
         - Daca sunt >12 produse, verifica ca apar butoanele de paginare
         - Testeaza Next, Prev, click pe numar de pagina
         - Testeaza ellipsis (...)

[ ] 2.8  Pagina 404
         - Acceseaza o ruta inexistenta ex: /xyz123
         - Asteptat: pagina NotFound cu link inapoi acasa

BLOC 4 — COS DE CUMPARATURI
================================================================================

[ ] 4.1  Vizualizare cos
         - Mergi la /cart
         - Verifica: lista produse, cantitati, preturi, subtotal

[ ] 4.2  Loading skeleton
         - La prima incarcare (refresh pagina) verifica animatia skeleton
           inainte sa apara produsele

[ ] 4.3  Stergere produs din cos
         - Apasa butonul "Sterge" pe un produs
         - Verifica ca dispare din lista si subtotalul se recalculeaza

[ ] 4.4  Undo (Command pattern)
         - Dupa stergere, apasa butonul "Undo"
         - Asteptat: produsul reapare in cos

[ ] 4.5  Cos gol

[ ] 4.6  Link spre Checkout
         - Cu produse in cos, apasa "Continua spre checkout"
         - Asteptat: redirectat la /checkout (sau /login daca nu esti autentificat)


================================================================================
BLOC 5 — CHECKOUT (wizard 5 pasi)
================================================================================

[ ] 5.1  Pas 1 — Contact
         - Verifica ca datele din profil sunt pre-completate
         - Testeaza validare: lasa firstName gol -> butonul "Continua" dezactivat
         - Completeaza toate campurile obligatorii (*)

[ ] 5.2  Pas 2 — Adresa
         - Completeaza strada, oras, cod postal
         - Testeaza validare: lasa strada goala -> "Continua" dezactivat
         - Schimba Regiune fiscala (EU / US / Asia) si verifica ca se salveaza

[ ] 5.3  Pas 3 — Livrare
         - Selecteaza Standard (15 MDL) si verifica estimarea costurilor
         - Selecteaza Express (35 MDL) si verifica ca totalul se actualizeaza
         - Verifica ca taxa regionala apare corect (Abstract Factory)

[ ] 5.4  Pas 4 — Plata
         - Testeaza selectia fiecarei metode: Card / PayPal / Stripe / Crypto
         - Introdu un cod promotional valid (ex: SUMMER20) si verifica acceptarea
         - Lasa campul gol si verifica ca merge mai departe

[ ] 5.5  Pas 5 — Review
         - Verifica ca toate datele introduse apar corect in sumar
         - Verifica totalul final (subtotal + taxa + livrare)
         - Adauga o nota optionala

[ ] 5.6  Plasare comanda
         - Apasa "Plaseaza comanda"
         - Asteptat: redirectat la /checkout/confirm/{orderId}
         - Cosul trebuie sa fie gol dupa confirmare

[ ] 5.7  Navigatie inapoi
         - Testeaza butonul "Inapoi" in fiecare pas
         - Verifica ca datele introduse se pastreaza

BLOC 6 — CONTUL MEUS
================================================================================

[ ] 6.1  Profil — vizualizare
         - Mergi la /account/profile
         - Verifica: initiale avatar, nume, email, rol (badge)

[ ] 6.2  Editare profil
         - Apasa "Editeaza"
         - Modifica firstName si/sau lastName
         - Apasa "Salveaza"
         - Asteptat: toast succes, header se actualizeaza cu noul nume

[ ] 6.3  Schimbare parola

         - Introdu confirmare corecta -> apasa "Schimba parola"
         - Asteptat: toast succes
         - Verifica: poti face login cu parola noua

[ ] 6.4  Navigare rapida din profil

         - Click pe "Wishlist" -> mergi la /account/wishlist

[ ] 6.5  Lista comenzi
         - Mergi la /account/orders
         - Verifica ca apare comanda plasata la Bloc 5
         - Verifica: data, total, status

[ ] 6.6  Detaliu comanda
         - Click pe o comanda
         - Verifica: produse, adresa livrare, metoda plata, total detaliat

[ ] 6.7  Wishlist
         - Mergi la /account/wishlist
         - Verifica produsele adaugate la Bloc 3.4
         - Apasa "Muta in cos" -> produsul trebuie sa apara in cart
         - Apasa "Sterge" -> produsul dispare din wishlist

BLOC 7 — ADMIN PANEL (necesita cont cu rol Admin)
================================================================================

[ ] 7.1  Acces protejat
         - Cu cont Customer, acceseaza manual /admin
         - Asteptat: redirectat (403 sau /)

[ ] 7.2  Dashboard
         - Logheaza-te cu cont Admin si mergi la /admin
         - Verifica cele 4 carduri: Vanzari azi, Comenzi noi, Stoc critic, Utilizatori noi
         - Verifica link-urile rapide spre Produse / Comenzi / Utilizatori

[ ] 7.3  Gestionare produse — listare
         - Mergi la /admin/products
         - Verifica ca produsele apar in tabel cu imagine, pret, stoc

[ ] 7.4  Cautare produs
         - Scrie in search bar un nume de produs
         - Verifica ca lista se filtreaza

[ ] 7.5  Creare produs nou
         - Apasa "Produs nou"
         - Completeaza: nume, descriere, pret, stoc, categorie
         - Salveaza -> produsul trebuie sa apara in lista si in catalog

[ ] 7.6  Editare produs
         - Click pe iconita edit la un produs
         - Modifica pretul
         - Salveaza -> verifica ca pretul nou apare in catalog

[ ] 7.7  Stergere produs
         - Click pe iconita stergere
         - Confirma dialogul
         - Verifica ca produsul dispare din lista

[ ] 7.8  Gestionare comenzi
         - Mergi la /admin/orders
         - Verifica ca apar toate comenzile

[ ] 7.9  Schimbare status comanda
         - Pe o comanda cu status "Pending", schimba la "Processing"
         - Verifica ca se actualizeaza in lista
         - Testeaza toate statusurile: Pending -> Processing -> Shipped -> Delivered

[ ] 7.10 Gestionare utilizatori
         - Mergi la /admin/users
         - Verifica ca apare lista de useri cu email si rol

[ ] 7.11 Cautare utilizator
         - Cauta dupa email
         - Verifica filtrarea

[ ] 7.12 Schimbare rol utilizator

         - Verifica ca se actualizeaza in lista
         - Schimba inapoi la Customer

BLOC 8 — SECURITATE SI EDGE CASES
================================================================================

[ ] 8.1  Rute protejate fara autentificare

         - Fa o actiune (ex: adauga in cos)

[ ] 8.5  Checkout cu cos gol
         - Goleste cosul, apoi incearca sa accesezi /checkout direct
         - Asteptat: redirectat inapoi la catalog sau mesaj de eroare

[ ] 8.6  Toast notifications
         - Verifica ca toast-urile apar si dispar automat dupa ~4 secunde
         - Testeaza toast succes (verde), eroare (rosu), info (albastru)

[ ] 8.7  Responsive / mobil
         - Redimensioneaza fereastra la latime mica (<768px)
         - Verifica meniu hamburger in header
